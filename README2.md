# Day 166 – File Search Tool

A Python script that searches for a file inside a folder and its subfolders.

## Description
This project allows users to search for a specific file by name.
It scans the entire directory recursively and shows the file path if found.

## Features
- Recursive search
- Works with subfolders
- Simple and useful utility

## Technologies Used
- Python
- os module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python file_search.py

4. Enter folder path and file name

## Example

Input:
folder: /Users/example/Documents  
file: notes.txt  

Output:
File found at:
/Users/example/Documents/Work/notes.txt
